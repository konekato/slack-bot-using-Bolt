# Slack bot using Bolt
Bolt for Python を用いた Slack ボットアプリケーションです。

## セットアップ
### このリポジトリをクローン
```bash
git clone git@github.com:konekato/slack-bot-using-Bolt.git
```

### Slack アプリの設定
[Bolt 入門ガイド](https://slack.dev/bolt-python/ja-jp/tutorial/getting-started)を参考に以下を行う。
1. Slackアプリの作成
1. トークンとアプリのインストール
1. プロジェクトをセットアップする
1. イベントを設定する

### 起動
```bash
python3 app.py
```
